Add the widget to your site, or use the shortcode **[powr-weather]** to add Weather to any post.

POWr plugins can be used on virtually any website!  Learn more at [www.POWr.io](http://www.powr.io/)

== Installation ==
To install POWr Weather
<br><br>
1. Upload 'powr-weather.php' to the '/wp-content/plugins/' directory
<br>
2. Activate the plugin through the 'Plugins' menu in WordPress
<br><br>
To add POWr Weather as a widget, go to **Appearance->Widgets**, and drag POWr Weather to where you'd like it to appear.
<br><br>
To add Weather within a post or a page, simply write the shortcode **[powr-weather label='MY LABEL']** (including the brackets) where you'd like your Weather to appear. Then visit your page!
<br><br>
Want a more detailed tutorial? Visit [How to add Weather to your Wordpress site](http://www.powr.io/tutorials/how-to-add-weather-plugin-to-your-wordpress-site)


== Frequently Asked Questions ==
= How can I add Weather to a blog post or page? = Just add the shortcode **[powr-weather label='MY LABEL']** (including the brackets) in your post where you'd like your Weather to appear. Then visit your page!

= How can I add  Weather to my site multiple times? = Just be sure to specify different label text, eg **[powr-weather label='MY LABEL 1']**, **[powr-weather label='MY LABEL 2']** for each time you use the plugin

= Why do you require me to sign up? = Signing up is what allows us to track which plugins are yours, and allows you to edit them right in your page.

= Will my users see the settings icon in the corner of the Weather? = No! You only see the settings icon because you own the plugins. To temporarily hide it, click on your page and then type 'p + down arrow' (the letter p followed by the down arrow). You can show the icon again by typing 'p + up-arrow'.

= I can't see the settings icon anymore, what happened? = You are just logged out of POWr.io. You can show the icon again by typing 'p + up-arrow'.
